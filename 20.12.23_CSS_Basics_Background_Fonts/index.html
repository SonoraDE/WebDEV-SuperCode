Drei Möglichkeiten CSS im HTML einzubinden

1. Extern

Im Head kommt der link mit CSS Pfad rein zum anknöpfen der CSS Datei

2. Intern

Wir können im "Head" per Style Element internes CSS ausführen.

Ist nicht empfehlenswert aufgrund von schlechterer Übersicht. HTML soll HTML bleiben und CSS soll CSS bleiben.

3. Inline Style

Wir können zum Beispiel einem H3 einen Style zuweisen und somit inline stylen

Auch das wird nicht empfohlen, da man HTML und CSS vermischt

Um mehrere Attribute zu verwenden trennen wir mit einem ; ab nach dem Attribut.
